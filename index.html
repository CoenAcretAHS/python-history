Weather Data Clouds Guides

Hi there, welcome to a website about weather!

This website has a number of features, such as a tool for finding the weather by co-ordinates, all over the world. There is also a page for identifying different types of clouds, via an image carousel. This site uses JavaScript, CSS and HTML (HyperText Markup Language) to bring these wonders to you. You can see this by pressing F12 below are some reference materials that may assist you in viewing this site - there is a storyboard, reflection statement and a user guide linked below.
